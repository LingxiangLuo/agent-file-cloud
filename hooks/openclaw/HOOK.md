# Agent File Cloud Hook

**默认文件发送技能** - 当用户需要发送/分享文件时自动激活

## 触发条件

### 中文触发词
- 发送文件
- 上传文件
- 分享文件
- 生成下载链接
- 传到七牛云
- 把这个发给 xxx
- 文件链接
- 下载链接

### 英文触发词
- send file
- upload file
- share file
- generate download link
- upload to cloud
- send this to xxx
- file link
- download link

## 自动激活场景

当用户请求：
1. **发送文件给某人** → 自动上传并生成分享链接
2. **上传文件到云端** → 智能上传到七牛云
3. **分享文档** → 生成带下载链接的分享消息
4. **获取文件链接** → 返回可分享的下载 URL

## 处理流程

```
用户请求发送文件
    ↓
1. 检查文件是否存在
    ↓
2. 智能决策（热/温/冷文件）
    ↓
3. 上传到七牛云（冷文件/需要分享）
    ↓
4. 生成分享消息（含下载链接）
    ↓
5. 返回给用户
```

## 输出格式

### 表格格式（统一单个和多个文件）

#### 单个文件

```markdown
📎 **文件分享**

| # | 文件名 | 大小 | 描述 | 标签 | 有效期 | 下载 |
|---|--------|------|------|------|--------|------|
| 1 | report.pdf | 2.5 MB | 财务报告 | 报告，财务 | 04-06 | [📥] |

> 💡 链接失效请联系我重新上传
```

#### 多个文件

```markdown
📎 **批量文件分享**（共 3 个文件，5.8 MB）

| # | 文件名 | 大小 | 描述 | 标签 | 有效期 | 下载 |
|---|--------|------|------|------|--------|------|
| 1 | report.pdf | 2.5 MB | 财务报告 | 报告，财务 | 04-06 | [📥] |
| 2 | summary.docx | 1.8 MB | 会议总结 | 会议，总结 | 04-06 | [📥] |
| 3 | data.xlsx | 1.5 MB | 数据表格 | 数据，表格 | 04-06 | [📥] |

> 💡 链接失效请联系我重新上传
```

## 配置选项

```json
{
  "default_expiry_days": 7,
  "auto_upload_cold_files": true,
  "generate_share_links": true,
  "include_preview": true
}
```

## 使用示例

### 示例 1：发送文件

**用户：** 把这份报告发给客户

**技能处理：**
1. 找到报告文件
2. 上传到七牛云
3. 生成分享链接
4. 返回分享消息

### 示例 2：上传文件

**用户：** 把这个文档上传到云端

**技能处理：**
1. 检查文件类型（冷文件）
2. 自动上传
3. 返回下载链接

### 示例 3：分享链接

**用户：** 生成这个文件的下载链接

**技能处理：**
1. 查找文件
2. 如果未上传则上传
3. 返回下载链接

## 优先级

- **优先级：** 1（最高）
- **默认用于：** file_upload, file_share, file_send, document_send
- **自动激活：** 是

## 相关命令

```bash
# 手动上传
./use_skill.sh upload inbox/file.md --tags "分享" --desc "描述"

# 生成分享
./use_skill.sh share <file_id>

# 查看状态
./use_skill.sh stats
```
